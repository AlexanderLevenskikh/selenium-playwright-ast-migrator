# Agent skills

Small composable behavior contracts for the standard migration flow. Installed skills: `plow-ahead`, `read-the-damn-docs`, `agent-watchdog`, `efficient-frontier`, `quick-recap`, `plan-arbiter`, and `root-cause-prioritization`. They never override scope, permissions, verification, or final gates. Skill-use recording is optional diagnostics; it is not required for a run to pass and must not become a second evidence workflow.
