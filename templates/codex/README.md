# Codex helper files

These files are optional. They are meant for small, bounded Codex tasks inside the migration kit.

Recommended use:

```text
Read migration/codex/CODEX.md and migration/codex/prompts/ticket-fix-prompt.txt.
Fix only the current ticket.
```

For review:

```text
Read migration/codex/CODEX.md and migration/codex/prompts/review-prompt.txt.
Review the current diff.
```
