# Selenium Playwright Migrator 0.0.0-preview.1

First public preview release.

## Stable path

- Selenium C# / NUnit or xUnit source migration toward Playwright .NET / NUnit or xUnit.
- Migration reports, quality gates, verification reports, PR packs, evidence packs, and public playground assets.
- Migration kit bootstrap for guarded agent-assisted runs.

## Experimental preview paths

These are intentionally not marketed as production-ready:

- Playwright TypeScript target output.
- Selenium Java source parsing.
- Selenium Python source parsing.

## Install

```bash
dotnet tool install --global SeleniumPlaywrightMigrator --prerelease
```

## Notes

Validate generated output with the normal reports and target project checks before accepting migrations.
